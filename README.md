# YouTube Transcription AI

This is a quick example of how to use OpenAI Whisper to transcribe videos to text. 

This can be used in content creation to automate research and extract ideas from video. 

This script will download a video from YouTube and generate a transcript of the video. 

## Getting Started

```
python3 main.py https://www.youtube.com/watch\?v\=RPVxwzJtp4o
```
